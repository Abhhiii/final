1. create repo in github
2. open a folder in vscode with desired name
3. in vscode terminal switch to git-bash
4. in terminal git init .
5. git remote add origin <repo_url>
6. git branch -M master
7. git add .
8. git commit -m "message"
9. git push origin master
10. when done create new branch and start working on that , git checkout -b <branch_name>
11. when switching to older repo, git checkout <branch-name>
12. git pull
